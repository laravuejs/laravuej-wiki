### Utils
- **Arquivo CheckPermission.js**
```
Esse arquivo utiliza o usePage do inertiajs para recuperar o usuário logado, e recebe como parametro a permissão e válida se o usuário possui permissão. Essa função é funcional utilizando a biblioteca de permissões do Spatie. Da forma que a função foi contruida é possivel passar no parametro uma string ou um array de permissões para serem validadas.
```
